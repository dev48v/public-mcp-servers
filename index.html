🔌 MCP Directory

A map of 100+ hand-curated Model Context Protocol servers (with connect configs) plus a 18,496-server catalog from every public registry — what they do, what they cost, and exact configs to connect them to Claude Code, Claude Desktop, Cursor or VS Code.

MCP ≠ API. An API is a service's raw interface. An MCP server is a standard connector that wraps tools/data so any AI agent can use them — the agent speaks one protocol (MCP) instead of a hundred different APIs. Server code here is almost always free & open source; the tier badge rates what it costs to actually use it end-to-end, including the service behind it.
Free = works end-to-end at $0 · Freemium = free tier exists, real usage needs a paid plan/API key · Paid = needs a paid product or paid API to be useful
⚠️ Security: MCP servers run with YOUR credentials and can act on your accounts. Prefer official servers, read community code before connecting, scope tokens narrowly, and treat fetched web content as untrusted (prompt injection).
